Installing and compiling IPL98

The library can be used as a C only library or as a C++ library. In both cases place the IPL 98 files on your local drive, say c:/ipl98/, in exactly the same structure as you received the IPL98 files in. Then make sure your compiler have a path set to this place, ie. c:/ipl98/. To make the library easy to use only two files are needed in a project, these files take care of including all other files in the library. Before explaining how to include the library you have to be aware of the hardware you are using.
