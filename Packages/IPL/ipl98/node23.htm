Return values

All functions and methods which can result in errors returns a boolean value. This value can be used by the programmer to recover from the error. A simple example is given here: 
CImage Img;
if (Img.Load(``c:/temp/doggie.bmp''))
{
   // do some image processing
}
else
{
   cout << ``Could not find the file c:/temp/doggie.bmp'' << endl;
}
This example prevents the image processing code to start on an empty object.
