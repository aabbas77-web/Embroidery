Basic types

When working with images it is important to have some basic types with a well defined number of bytes in memory. This is due to the fact that the image file formats used on a computer are created with a specific number of bits to represent each pixel. Because of the lack of such well defined types in the C and C++ language it has been nessecary to define the following types:

Type name Description Min. value Max. value
UINT8  unsigned 8 bit integer 0 255
INT8  signed 8 bit integer -128 127
UINT16  unsigned 16 bit integer 0 65535
INT16  signed 16 bit integer -32768 32767
UINT32  unsigned 32 bit integer 0 4294967295
INT32  signed 32 bit integer -2147483648 2147483647
FLOAT32  32 bit floating point 1E-37 1E+37

The naming convention is ``U'' for unsigned otherwise signed, followed by the type (integer or float) and at the end the number of bits used to represent the number.
