Macros for small usefull functions

Two files in the library contains small functions which are usefull in many places when programming. The files are ipl98/kernel_c/general_functions/ipl_general_functions_c_only.h and ipl98/cpp_classes/general_functions/ipl_general_functions.h. Here follows a description of the C++ macros only, they are all template based. For the involved type T it is written what operator need to be implemented to make the macro work with the type:

  • PI  This is simply the definition of PI ( ), not a template macro.
  • PI2  Defined as 2*PI. Not a template macro.
  • Sqr(x)   Calculates the square of x (x*x). The *-operator need to work for the type of x.
  • Swap(x,y)   Swaps the contents of x and y. The =-operator need to work for the type of x and y.
  • Max(x,y)   Returns the greatest value of x and y. The >-operator need to work for the type of x and y.
  • Min(x,y)   Returns the minimum value of x and y. The <-operator need to work for the type of x and y.
  • Round(x)   Round x to nearest integer value. The (<,+,-)-operators need to work for the type of x.
  • RoundUINT8(x)    Round x to nearest UINT8 value or if x>255 then x=255. The (<,>,+)-operators need to work for the type of x.
