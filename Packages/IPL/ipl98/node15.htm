Using the C++ part of the library

In case of using the C++ part of the library add the files /ipl98/ipl98_cplusplus.h and /ipl98/ipl98_cplusplus.cpp in your project. These files include all other files needed. Then include the file /ipl98/ipl98_cplusplus.h in your own files where you want to use the library.

A special namespace  called "ipl" is used to avoid name clash with other libraries. If you are creating an application where none of the classes/macros/types in IPL98 interfere with your environment you can "skip" the namespace by writing this in you files: 

     #include <ipl98/ipl98_cplusplus.h>
     using namespace ipl;
An object of say type CImage is then declared by: 

     CImage MyImage;
If name clash is a problem you just do like this: 

     #include <ipl98/ipl98_cplusplus.h>
     ipl::CImage MyImage; /* example: declaration of a variable of type CImage */
You can always try the first solution to see if you have any problems with name clash.
