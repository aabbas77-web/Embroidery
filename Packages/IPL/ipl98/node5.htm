IPL98 version 1.10

Version 1.10 became available the 20th of december 1999. The following features were added to the container related part:
  • CComplexImage  A new container class, contains pixels of type float represented by four bytes (FLOAT32), each position has a real and an imaginary value. Can be saved in an ASCII format including the history.
  • Arithmetic methods  contained in all container classes:
    • ApplyBias(int Bias) 
    • ScaleIntensity(float Prefactor) 
    • Add(CStdImage &Addend) 
    • Subtract(CStdImage &Subtrahend) 
    • Multiply(CStdImage &Factor) 
  • Other methods added to the container classes:
    • CopyDisplace(int dx,int dy) 
    • CutOrZeroPad(Source) 
    • Copy(Source) 

The following features are added to the algorithm part:

  • CLocalOperation  simple transformations depending only on the single pixel values. For instance conversions between different image formats which involves scaling of values.
  • CMaskOperation  simple mask and filter operations. Methods available in the class are:
    • DrawHistogram(...)  
    • EqualizeHistogram(...) 
    • NonlinearCopyToByte(...) 
  • CSpectral     fourier and related transforms. Methods available in the class are:
    • FFTForward(CComplexImage &Img)
    • FFTInverse(CComplexImage &Img)
  • CStatistic  calculates minimum, maximum, mean, standard deviation and a histogram  for a given image. Methods available in the class are:
    • GetMean()  
    • GetMin()  
    • GetMax()  
    • GetStdDev()  
    • GetSizeOfHistogram()
    • DeriveStats(const CFloatImage& Image)
    • DeriveStats(const CIntImage& Image)
    • DeriveStats(const CStdImage& Image)
    • DeriveStats(const CPixelGroup& PixelGroup)
